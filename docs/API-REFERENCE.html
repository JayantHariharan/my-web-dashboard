POST /api/auth/login Authentication

Authenticate a user and return user details. Uses constant-time comparison to prevent timing attacks.

Request Body

FieldTypeConstraintsExample
usernamestring1-100 chars, must contain at least one letter"player123"
passwordstring1-100 chars"SecurePass123!"
Request Example
{
  "username": "player123",
  "password": "SecurePass123!"
}

Response

Success (200 OK)

Response Example
{
  "message": "Login successful",
  "username": "player123",
  "user_id": 1
}

Error Responses

StatusDescription
400 Bad RequestMissing or invalid credentials
401 UnauthorizedInvalid username/password (generic message)
429 Too Many RequestsRate limit exceeded (20 attempts per hour)
500 Internal Server ErrorDatabase or server error

Security Features

  • Constant-time comparison: Prevents timing attacks
  • Generic errors: Doesn't reveal if username exists
  • IP logging: Tracks last_login_ip and last_login_at
  • Rate limiting: 20 requests per hour per IP (shared with signup)
POST /api/auth/signup Authentication

Register a new user account. Checks for duplicate usernames and stores password securely with bcrypt + pepper.

Request Body

FieldTypeConstraintsExample
usernamestring3-100 chars, must contain letters"newplayer123"
passwordstringMin 8 characters"SecurePass123!"
confirm_passwordstringMust match password"SecurePass123!"
Request Example
{
  "username": "newplayer123",
  "password": "SecurePass123!",
  "confirm_password": "SecurePass123!"
}

Response

Success (200 OK)

Response Example
{
  "message": "Signup successful",
  "username": "newplayer123",
  "user_id": 2
}

Error Responses

StatusDescription
400 Bad RequestValidation error (empty fields, password mismatch, weak username)
409 ConflictUsername already exists
429 Too Many RequestsRate limit exceeded (20 attempts per hour)
500 Internal Server ErrorDatabase or hashing error

TEST Quick cURL Examples

Auth

# Signup
curl -X POST http://localhost:8000/api/auth/signup \
  -H "Content-Type: application/json" \
  -d '{"username":"test","password":"Test1234","confirm_password":"Test1234"}'

# Login
curl -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"test","password":"Test1234"}'

# Get profile
curl "http://localhost:8000/api/auth/me?username=test"

Apps

# List apps
curl http://localhost:8000/api/apps/

# Launch app
curl -X POST "http://localhost:8000/api/apps/1/launch?username=test"

Games

# List games
curl http://localhost:8000/api/games/

# Submit score
curl -X POST "http://localhost:8000/api/games/tic-tac-toe/scores?username=test" \
  -H "Content-Type: application/json" \
  -d '{"score":1500,"metadata":{"won":true}}'

# Get leaderboard
curl "http://localhost:8000/api/games/tic-tac-toe/leaderboard?limit=10"

Health

curl http://localhost:8000/health
