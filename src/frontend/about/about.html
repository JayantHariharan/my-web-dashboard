About this project

PlayNexus is an experimental portal: sign in once, then move between Games, Apps (news, media tools, resume helpers, and more as we ship), and Community spaces. The stack is deliberately small so it runs comfortably on Render and Supabase free tiers.

The command hub after login is your home base. Use the top navigation to switch sections, or open full pages like the Play Center when you want a focused layout.

TECH STACK

  • Backend: FastAPI, Python 3.12+
  • Frontend: static HTML, CSS, vanilla JavaScript
  • Database: SQLite locally; PostgreSQL (Supabase) in production
  • Auth: session in the browser today; token-based auth is on the roadmap

Project structure, environment variables, and hooks are documented in README.md at the repository root. Replace placeholder GitHub or Discord links on the Community page with your real URLs when you go live.
